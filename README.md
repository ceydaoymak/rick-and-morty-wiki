# Rick and Morty Wiki

A simple wiki application built with **React** that allows users to explore characters, episodes, and locations from the **Rick and Morty** universe using real data from the public Rick and Morty API.

---

## Technologies
- React
- React Router
- Axios
- Tailwind CSS
- Vite
- TypeScript

---

## Features
- Home page with basic statistics
- Character, episode, and location listings
- Detail pages with related data
- Search and pagination
- Client-side routing

---

## API
Data is fetched from the public Rick and Morty API:  
https://rickandmortyapi.com/
